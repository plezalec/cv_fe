One of my jobs at ARSO ARSO: Agencija Republike Slovenije za okolje (Environmental Agency of the Republic of Slovenia).
vreme.arso.gov.si was to establish and maintain data stream processing pipelines for the latest MTG satellite data using PyTroll Pytroll: easy to use, modular, free and open source python framework for the processing of earth observation satellite data.
pytroll.github.io . This involved working with large volumes of data, ensuring its quality and reliability, and making it accessible for various applications.
One of the products is shown below. The animation is from the ARSO website, and it shows the latest geocolor satellite data for Slovenia. The animation is updated in real-time, and it provides valuable information for weather forecasting and monitoring in the region.

ARSO data viewer for GBL GBL: Gasilska brigada Ljubljana (Ljubljana Fire Brigade).
gasilskabrigadaljubljana.si

I was responsible for solo development of the ARSO and GBL data viewer, handling full-stack from design to deployment.

This web application allows users to visualize and analyze various datasets from ARSO. It provides an intuitive interface for exploring weather data, specifically designed for use by firefighters in cases of extreme weather conditions. The application includes features such as interactive maps, real-time data updates, and includes data from wide variety of sources.
